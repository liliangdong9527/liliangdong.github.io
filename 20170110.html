20170110 JavaScript复习总结

JavaScript 介绍

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML网页上使用，用来给HTML网页增加动态功能。

JavaScript 的特点

JavaScript中的数据类型分为两种：

    基本数据类型(Number,String,Boolean,Null,Undefined...)

    引用数据类型即对象类型(Object,Function...)

JavaScript中for循环的作用

for: 循环遍历数组

for(var i=0;i<2;i++){};

for in: 遍历非数组对象

var obj={};

for(var j in obj){};

关于JavaScript中的函数

JavaScript中函数的几种声明方式

注：函数是JavaScript的一等公民。

1、函数声明式声明

function fun(){

};

2、表达式声明

var foo = function (){

};

var fun1 = function fun2(){

};

3、匿名声明

(function fun(){

})();

4、动态声明

var fun3 = new Function("a=1","b=2","return a+b");

函数中的参数

函数的参数分为实参和形参

function fun(a,b,c){//a,b,c是形参

}

fun(1,2,3)//1,2,3是实参

函数参数也可以输函数名

function foo(callback){

callback();

}

添加事件侦听的方法

Obj.onclick = fun;//常用的添加方法

function fun(e){

console.log(e.target)

}

Obj.addEventListener("mousedown",fun,false);//W3C推荐 注意浏览器兼容

关于DOM和BOM

DOM 一般所说的DOM指的是HTML DOM，也就是W3C的DOM Document Object Model

BOM 各个浏览器不一,Window对象~=类

this 的指向

1、this指向标签

2、this指向Window

function foo(){

console.log(this);

}

foo();

3、this指向Object对象

var obj = {};

obj.fun = function (){

console.log(this);

}

obj.fun();

改变函数this的指向方法

Function.apply(Object [ , argArray]);

Function.call(Object [ , arg1[, arg2 ...]]);

JavaScript 的渲染顺序

自上到下，从左到右,即声明同一个函数名的函数，调用时最下层的函数会覆盖上层的函数

声明式和表达式共存时，声明式会自动提升到表达式前
