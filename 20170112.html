20170112 内置对象和touch事件

内置对象

在js里，一切皆为或者皆可以被作为对象。可通过new一个对象或者直接以自变量形式创建变量。所有变量都有对象的性质。

数组对象Array

数组是值的有序集合，数组中可以存放数值，也可以存放字符串。

数组的创建可以通过 var arr = [1,2,3,4,...] 。

数组的索引是通过数组的下标arr[index] 来获得且第一个的下标是0，数组的长度通过arr.length 属性获得。

数组内元素的添加/删除方法

push(): 将新元素添加到一个数组中，并返回数组的新长度值。

pop(): 移除数组中的最后一个元素并返回该元素。

shift(): 移除数组中的第一个元素并返回该元素。

数组的其他常用方法

reverse(): 返回一个元素顺序被反转的 Array 对象。

sort(): arr.sort(function(a,b){return b-a;}) JS数组自带排序方法(从小到大排序)。

错误对象Error

JS解析成功或执行时，一旦发生错误，引擎就会抛出一个错误对象，所有抛出的错误都是Error构造函数的实例。

Error构造函数接受一个参数，表示错误提示。 var err = new Error('This is Error!'); err.message;

注意: 根据语言标准，Error对象的实例必须有message属性表示出错时的提示信息，官方没有提及其他任何属性。大多数浏览器对Error实例还提供name和task属性，分别表示错误的名称和错误的堆栈，但它们是非标准的，不是所有浏览器都有。

内置错误类型(派生对象)

1、ReferenceError-引用错误

2、RangeError-范围错误

3、SyntaxError-语法错误

4、TypeError-类型错误

5、URIError-URI错误

处理错误

try{

throw "发现错误！";

}catch (e){

console.log(e.message);

}finally{

//无论有无错误都要执行的代码块

}
